bulma-wysiwyg work in progress

tiny wysiwyg rich text editor for Bulma
based on bootstrap-wysiwyg

About

This tiny (5 KB, < 100 lines) pure Javascript Bulma plugin turns any TEXTAREA into a WYSIWYG rich-content editor, inspired by bootstrap-wysiwyg. Here are the key features:

  • Does not force any styling - it's all up to you
  • Uses standard browser features, no magic non-standard code
  • Does not create a separate frame, work over divs etc - instead keeps it simple and saves the content on your TEXTAREA

Why?

While building a project with Bulma, we found plenty of good HTML5 WYSIWYG editors online, but no one was meant to work with Bulma.


To-do

bulma-wysiwyg is a very early-stage project. Any contributions are welcome, however the main points as of now are:
